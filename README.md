# youtube-summarizer-docker-image

Before starting, make sure Docker is installed

## Compose docker Image:

### Linux
```bash
sudo docker compose up -d
```

### Windows

```bash
docker-compose up -d
```

## Test
- Navigate to http://localhost:3333 in the browser
